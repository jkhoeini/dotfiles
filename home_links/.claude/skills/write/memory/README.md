# Writing Memory

These files help Claude learn your writing preferences over time.

## Files

| File | Purpose |
|------|---------|
| `patterns.md` | Writing patterns learned from your documents |
| `stakeholders.md` | Key people and their communication preferences |
| `corrections.md` | Writing corrections to avoid repeating |

## Exemplars

The `exemplars/` folder contains reference documents for few-shot prompting:

| File | Use For |
|------|---------|
| `two-pager-exemplar.md` | Two-pager format documents |
| `strategy-exemplar.md` | Strategy docs, pre-reads |
| `proposal-exemplar.md` | Proposals, initiatives |

## How It Works

After writing documents, Claude may suggest updates to these files based on:
- Feedback you give on drafts
- Patterns in successful documents
- Explicit corrections ("don't use that phrase")
