# Write Skill - Slack & Email Templates

Quick reference templates for common communication patterns.

## Slack Templates

### Ask
```
TL;DR: <ask> | Context: <1-2 lines> | What I need: <thing> by <date>
```

### Decision
```
Decision needed by <date>: <decision> | Options: A) ... B) ... | Recommendation: <line>
```

### Status
```
TL;DR: <green/yellow/red> | Progress: <bullets> | Risks: <bullets> | Next: <bullets>
```

### Close-the-loop
```
Update on <topic>: <outcome>. Thanks to @person for <contribution>.
```

## Email Templates

### Subject Prefixes

| Prefix | When |
|--------|------|
| [ACTION] | Needs recipient action |
| [DECISION] | Needs approval/choice |
| [FYI] | Informational only |
| [BLOCKED] | Waiting on something |

### [ACTION] Template
```
Subject: [ACTION] <what> by <date>

<Ask in first sentence>

Context:
- <why this matters>
- <what you've already done>

Next step: <specific action> by <date>
```

### [DECISION] Template
```
Subject: [DECISION] <topic> - response needed by <date>

<Decision statement>

Options:
A) <option> - <tradeoff>
B) <option> - <tradeoff>

Recommendation: <which and why>

Please reply with your choice by <date>.
```

### [FYI] Template
```
Subject: [FYI] <topic>

<Key takeaway in first sentence>

Details:
- <point 1>
- <point 2>

No action needed. Reply if questions.
```

## Quality Checklists

### Slack
- Single message (not wall of text)?
- Thread-safe (can be read without context)?
- Scannable (key info visible without scrolling)?
- Mentions justified (@person only if they need to act)?

### Email
- Intent in subject line?
- One topic per email?
- Ask + deadline in first 2 lines?
- Skimmable structure (bullets, bold key points)?
